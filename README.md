# Proyecto: Innovación Colaborativa

## Contexto General

La Universidad Tecnológica **"Nuevo Horizonte"** desarrolla el proyecto piloto **Innovación Colaborativa**, cuyo propósito es crear un entorno de trabajo digital que facilite la colaboración efectiva entre docentes y estudiantes en la creación de materiales educativos digitales.

El proyecto es ejecutado por un equipo multidisciplinario conformado por **seis integrantes**, cada uno con roles claramente definidos:

* Coordinador del proyecto
* Diseñador instruccional
* Editor de contenido
* Programador web
* Especialista multimedia
* Revisor

Debido a que el proyecto se desarrolla **100 % en modalidad remota**, el equipo requiere herramientas y prácticas que permitan:

* Organizar y asignar tareas de manera eficiente.
* Compartir y gestionar archivos de forma segura.
* Trabajar de manera colaborativa y en tiempo real.
* Proteger la información y los datos del proyecto.
* Garantizar el reconocimiento de la autoría y el respeto a la propiedad intelectual del material producido.

## Objetivo del Equipo

El objetivo principal del equipo es **analizar, configurar y recomendar** las mejores herramientas digitales y buenas prácticas que aseguren una colaboración:

* Eficiente
* Segura
* Ética
* Legalmente responsable

Este repositorio documenta las decisiones, configuraciones y recomendaciones adoptadas para el desarrollo exitoso del proyecto **Innovación Colaborativa**.
